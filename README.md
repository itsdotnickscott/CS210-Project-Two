# CS210-Project-Two

## Summary
This project is an Interest Calculator which takes in user's input for initial balance, monthly deposits, annual interest, and the number of years. The calculator will calculate interest earned after each year for the given timeframe.

## Strengths
The project was designed with the user heavily in mind. Therefore, the user experience should be smooth and seamless. Tables and menus are organized in an eye-catching way. The code feels organized and well=documented.

## Improvements
main.h contents would be moved to main.cpp as function declarations, as it is non-traditional. The table starts to become formatted in a wack way if the numbers get exceptionally large, therefore a function to handle large numbers would be beneficial to the user.

## Challenges
Formatting everything took a while to look nice. cppreference.com was a great resource for me to understand the format literals.

## Skills
Project and class design will be very important in the future. Being able to take specifications and converting it to actual code is a very useful skill in any aspect of programming.

## Readability
I made sure each function has a header comment which can be used to understand its purpose, what parameters are asking for, and what is returned, if any. Each line is thouroughly commented so that each function and variable is easily understood to any who will edit the program in the future.
